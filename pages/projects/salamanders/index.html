Salamanders

plugins that live in the fire

Salamanders are plugins for LlizardOS - small creatures that add functionality without changing the core. They load at boot, listen for events, and extend what the system can do. Named for the mythical creatures said to live in flames.

planned ~ your plugin here salamanders await their purpose

how plugins work

Each salamander is a shared library that implements a simple interface. LlizardOS discovers them at boot, loads them, and calls their hooks when events occur - media changes, button presses, connection events.

// salamander interface type Salamander interface { Name() string OnLoad(ctx Context) OnMediaUpdate(state MediaState) OnButtonPress(button Button) OnUnload() }

Drop a .so file in /usr/lib/salamanders, and it wakes up with the system.

the host

Salamanders exist within LlizardOS - the operating system that gives them a home and purpose.
